# FoodHub — Food Delivery Order Analysis

## Problem Statement
Analyze food delivery order data for FoodHub, a food aggregator company, to understand customer ordering behavior, restaurant performance, and delivery patterns, in order to support business decisions on operations and marketing.

## Approach
Performed exploratory data analysis covering missing value treatment, univariate analysis (distributions of order value, delivery time, ratings), and multivariate analysis (relationships between cuisine type, day of week, and order patterns). Derived revenue and delivery-time SLA metrics, and translated findings into business recommendations.

## Skills
Exploratory data analysis, univariate & multivariate analysis, business metric derivation, data-driven recommendations

## Tech Stack
Python, pandas, NumPy, Matplotlib, Seaborn

## Dataset
FoodHub order data, provided as part of PGP-AIML coursework.
