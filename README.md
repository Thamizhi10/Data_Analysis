# Data Analysis

A collection of exploratory data analysis and business analytics projects using Python and SQL to derive insights from real-world datasets.

## Projects

| Project | Task | Key Techniques |
|---|---|---|
| [FoodHub](./01-foodhub-eda) | Food delivery order analysis | EDA, univariate/multivariate analysis, business metrics |
| [New Wheels](./02-newwheels-sql-business-analysis) | Vehicle resale quarterly business report | SQL joins, window functions, KPI derivation |
| [E-news Express](./03-enews-express-ab-testing) | Landing page A/B test analysis | Hypothesis testing, t-test, chi-square, ANOVA |

## Tech Stack

`Python` · `pandas` · `NumPy` · `Matplotlib` · `Seaborn` · `SQL`
